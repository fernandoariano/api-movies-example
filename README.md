## How to run

**url:** http://localhost:8080/

### Start everything in containers:

```sh
make build run
```

### Start application on IDE and dependencies on containers:

```sh
make run-dependencies
```

**Run:** br.com.fs.api.movies.ApiMoviesExampleApplication.java
